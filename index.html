Simple.css

The simple, stupid CSS framework.

Simple.css is a bare-boned responsive CSS "framework" made to be customized. No fancy components, no JS plugins, no bloat. Simple.css is not made for the copy-paste web developer, it's for people who like to start with something simple and do the rest by themselves.

Download

Contribute, check out the source code on GitHub or just download the minified version from this website.

Download simple.min.css (15kb)

Installation

In order to use Simple.css you need to complete two simple steps:

  1. Save simple.min.css in your project's vendor/css folder.
  2. Include it in your project using <link rel="stylesheet" href="simple.min.css">

Note: If you didn't already know how to use a css framework then Simple is probably not for you.

The grid

There is no grid. Actually there is but not in the traditional sense of a predefined column number grid.
You do need a .container, a .row and at least a .col but the way you set the width, height and offsets using Simple is different.

There is no learning curve as long as you can remember 3 things:

  • w means width
  • h means height
  • There is a class for each division up to 12 for each property. Let me show you:

Grid example

Let's say I want two divs, each with one row. The first div will have 3 columns, all equal. The second div will have 2 columns, but I'd like the first column to be 3/5 of the total width.

Code:

<div class="container"> <div class="row"> <div class="col w1/3"> 1/3 width </div> <div class="col w1/3"> 1/3 width </div> <div class="col w1/3"> 1/3 width </div> </div> <div class="row"> <div class="col w3/5"> 3/5 width </div> <div class="col w2/5"> 2/5 width </div> </div> </div>

Output:

1/3 width
1/3 width
1/3 width
3/5 width
2/5 width

Half of something can be interpreted as 1/2, 2/4, 3/6, 4/8, 5/10 or 6/12. Any version you would like to use, it will work - you can also combine them.

Anything above 12 divisions will not work (ex: w1/14, w6/15).

Responsive

You can add responsive capabilities to your layout by simply adding some classes.

The default classes will apply to all screen sizes unless you specify overrides.

Remember two things:

  1. m means mobile
  2. t means tablet

If you want a div to take 50% width on desktop and tablet, but take 100% width on mobile devices:

<div class="col w1/2 m:w1/1">

Just prepend the targeted column size classes with m: or t: and you are ready to go.

Try resizing this window.


Removing offsets on mobile/tablet

If you set a general offset class (ex: .o1/5) and you want to remove it on mobile/tablet, you can use .m:o0 or .t:o0.

Typography

All font sizes are expressed in ems, you can change all at once by modifying the font size of the body or by brewing your own sizes. Here are the used font families:

body { font-family: "HelveticaNeue-Light", "Helvetica Neue Light", "Helvetica Neue", Helvetica, Arial, "Lucida Grande", sans-serif; } code, code * { font-family: Consolas, "Andale Mono WT", "Andale Mono", "Lucida Console", "Lucida Sans Typewriter", "DejaVu Sans Mono", "Bitstream Vera Sans Mono", "Liberation Mono", "Nimbus Mono L", Monaco, "Courier New", Courier, monospace; }

Utility classes

Simple has a few CSS utility classes that you can use:

  • .left will float an element to left
  • .right will float an element to right
  • .block will change the display of an element to block
  • .fill will change the width of and element to 100%
